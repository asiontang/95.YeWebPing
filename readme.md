# YeWebPing

## 项目初衷

开启VPN时,想知道 当前连接点 连接 常用网站 的 "连通性"如何? "访问速度"如何?

## APP 功能列表

1. 支持 Ping 指定域名的所有IP地址,统计出平均响应时间最快的IP.

1. 支持 HTTP.GET 指定域名,统计出平均响应时间最快的IP或域名.
